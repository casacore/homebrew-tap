# Casacore Tap

## How do I install these formulae?
`brew install casacore/tap/<formula>`

Or `brew tap casacore/tap` and then `brew install <formula>`.

## Documentation
`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
